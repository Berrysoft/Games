# Games
这里是一些使用VB+WPF重制的游戏。

## Snake
贪吃蛇，F5为单人游戏，F6为双人游戏，F7为双人竞争游戏。

## Square
俄罗斯方块，F5刷新。

## Maze
一个简单的迷宫游戏，F1自动走，F2给出下一步提示，F3给出路线，F5刷新。
